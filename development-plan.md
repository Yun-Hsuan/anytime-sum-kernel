# 長照聊天機器人開發計劃

## 專案目標
建立一個智能聊天機器人系統，用於：
1. 協助其他機構的照護人員或照護者家屬與醫院安寧照護團隊進行溝通
2. 自動蒐集照護者的病狀特徵
3. 根據收集的特徵為安寧照護團隊提供初步診斷建議
4. 降低醫院安寧照護團隊的日常對話工作量

## 系統架構設計

### 1. 資料模型設計
現有模型：
- 用戶（User）
- 項目（Item）

需要新增的模型：
- 病患資料模型（Patient）
- 症狀記錄模型（Symptom）
- 對話記錄模型（Conversation）
- 診斷建議模型（Diagnosis）

### 2. 用戶角色設計
需要擴展的角色：
- 照護人員（Caregiver）
- 家屬（Family）
- 醫療團隊（Medical Team）

### 3. API 端點設計
基於現有的 API 結構進行擴展
